## 📌 About

This is an interactive DSA learning roadmap tailored for:
- 🖥️ **Language:** C++
- ⏱️ **Time:** 1–2 hours/day
- 🎯 **Goal:** Deep understanding, not just problem grinding
- 📅 **Duration:** 26 weeks across 5 phases

Built by a learner, for learners — inspired by 10 years of MNC interview experience.

---

## 🗺️ Roadmap Phases

| Phase | Topic | Duration |
|-------|-------|----------|
| 1 | Arrays, Strings, Hashing, Binary Search | Weeks 1–4 |
| 2 | Linked Lists, Stacks, Queues, Recursion | Weeks 5–8 |
| 3 | Trees, Graphs, Heaps, Tries | Weeks 9–14 |
| 4 | Dynamic Programming | Weeks 15–20 |
| 5 | Greedy, Advanced Graphs, Mock Interviews | Weeks 21–26 |

---

## ✨ Features

- 📍 Phase-by-phase structured learning path
- 🔗 Curated LeetCode problem links with difficulty tags
- 💡 Expert tips for each topic
- 📅 Daily study structure per topic
- 🗓️ Weekly schedule template
- ⚡ C++ DSA cheatsheet with code snippets
- 🔖 Free resource links (NeetCode, Striver, CP-Algorithms)

## 🏃 Run Locally
```bash
# Clone the repo
git clone https://github.com/Indra-sekhar/dsa-roadmap.git

# Navigate into folder
cd dsa-roadmap

# Install dependencies
npm install

# Start development server
npm start
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
